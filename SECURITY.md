# Security Policy

## Supported versions

Security fixes are provided for the latest release branch.

## Reporting a vulnerability

Please report security issues privately:

- Email: security@codesfly.dev
- Include: impact, reproduction steps, affected files, suggested mitigation

Do **not** open public issues for unpatched vulnerabilities.

## Sensitive data policy

This repository provides architecture/templates only.
Do not commit personal identifiers, tokens, private chat logs, or secrets.
